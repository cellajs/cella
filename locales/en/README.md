# Locales

Here you can manage your locales per language. The basics are:

- `common.json`: translations that are required for the basics that cella provides. Used in frontend and backend.
- `about.json`: for marketing page translations
- `app.json`: extended `common` translation for app-specific texts.
- `backend.json`: backend-only translations
- `errors.json`: `error` translations used in both frontend and backend
